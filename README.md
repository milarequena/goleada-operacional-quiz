# goleada-operacional-quiz
Quiz competitivo em grupos para fixar os fluxos de acionamento da operação de forma leve e engajante. O material é dividido em 8 grupos, cada um com 5 a 6 perguntas entre múltipla escolha, verdadeiro/falso e perguntas abertas, cobrindo temas como Pack Ops, métricas, badges, dimensionamento e finais de semana.
